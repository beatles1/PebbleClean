### Pebble Watchface

![Screenshot](https://appstore-api.repebble.com/api/assets/screenshots/30059ade40e44f408302f543/basalt/c2e53947-0140-4d3f-8547-363cbba08eac.png)

Simple digital watchface with:
- Temperature from Open Meteo
- Bar showing percentage of health target achieved
- Bar showing battery level

Colours of bars are customisable via settings as well as the health metric and target value.

I created this watchface many years ago for my first smartwatch (a Time Steel) and have since ported updated versions to Amazfit, Fitbit and Garmin but back with an updated version of this for Pebble again.
